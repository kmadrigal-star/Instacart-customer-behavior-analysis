# Instacart Customer Shopping Behavior Analysis

## Overview
This project explores customer shopping behavior using Instacart order data to identify purchasing patterns, reorder trends, and peak shopping times.

## Data
- Multi-table Instacart dataset (orders, products, customers)
- Large-scale transactional data

## Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Reorder rate and frequency analysis
- Pivot tables and visualization

## Tools
- Python (Pandas, NumPy)
- SQL
- Excel
- Google Sheets

## Results
- Identified high-demand products and categories
- Determined peak ordering times
- Provided insights relevant to inventory planning and marketing

## Next Steps
- Segment customers by purchasing behavior
- Build predictive reorder models
